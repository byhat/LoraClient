# LoRaCore - передача данных по LoRa (E22-400T22U USB)

**LoRaCore** - это библиотека на C++/Qt6, реализующая передачу данных поверх USB-адаптера **Ebyte E22-400T22U** (LoRa модуль с UART-to-USB). Поддерживает фрагментацию пакетов, подтверждения приёма и повторные отправки.

## Особенности

- Передача пакетов **любого размера** (автоматическая фрагментация на чанки ≤26 байт).
- Надёжная доставка с поддержкой:
  - `ACK` на каждый чанк,
  - `PACKET_ACK` на весь пакет,
  - повторных отправок (до 5 попыток с таймаутом 1 с).
- CRC8 (Dallas/Maxim) для контроля целостности фреймов.
- Простой интерфейс: открыть порт → отправить/получить `QByteArray`.

## Компоненты

- `LoRaWorker` - высокоуровневый интерфейс (работа с портом, сигналы/слоты).
- `LoRaUsbAdapter_E22_400T22U` - реализация протокола (фреймирование, ACK, reassembly).

## Сборка

Требования:
- CMake ≥ 3.19
- Qt6 (Core + SerialPort)

```bash
mkdir build && cd build
cmake ..
cmake --build .
```
